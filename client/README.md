# Problem Statement
Fashion creators post outfits on social media, but followers have no direct way
to shop the exact items being worn.

Fashiongram solves this by allowing creators to tag clothing items directly on
images, enabling followers to click and shop instantly.

# Solution
Fashiongram is a MERN-based social platform where creators upload images and
tag clothing items with direct purchase links.

Each tag tracks engagement, helping creators understand which items perform best.

# Key features
- JWT Authentication (Login/Register)
- Image uploads with Cloudinary
- Clickable clothing tags on images
- Tag-level click analytics
- Creator-only analytics dashboard
- Follow / Unfollow users
- Like & Comment system
- Secure post & tag deletion
- Pagination-ready feed

# Tech Stack

Frontend: React (Class Components)
Backend: Node.js, Express
Database: MongoDB + Mongoose
Auth: JWT
Media: Cloudinary
